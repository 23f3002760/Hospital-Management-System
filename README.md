# Hospital Management System (HMS)

## Overview
The Hospital Management System is a web-based application designed to improve coordination between **Admins**, **Doctors**, and **Patients**.  
It helps in managing appointments, patient records, doctor schedules, and general hospital operations in a digital format instead of manual paperwork.

---

## Tech Stack
- **Backend:** Python (Flask)
- **Database:** SQLite using SQLAlchemy ORM
- **Frontend:** HTML, CSS (Bootstrap), Jinja2 Templates

---

## Key Features
- **Admin**
  - Add and manage doctors and patients
  - View and monitor all appointments
  - Maintain hospital records

- **Doctor**
  - View assigned patients and their medical history
  - Update diagnosis and treatment notes
  - Manage availability

- **Patient**
  - Book or cancel appointments
  - Search doctors by specialization
  - View past treatment history
